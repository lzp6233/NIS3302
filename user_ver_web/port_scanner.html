关于 WebPortScanner

WebPortScanner 是一个基于网页的网络端口扫描工具，旨在提供类似于 nmap 的功能，但具有更直观的图形界面。

安全与隐私

我们重视您的安全与隐私。所有扫描操作都在您的本地设备上进行，扫描结果不会被上传到任何服务器。

  • 扫描数据完全保存在本地浏览器中
  • 无需注册账号，保护您的个人信息
  • 符合隐私保护法规要求

技术特点

WebPortScanner 采用了先进的前端技术，提供流畅的用户体验和强大的功能。

  • 基于现代 Web 技术构建，无需安装客户端
  • 响应式设计，适配各种设备屏幕
  • 直观的图形界面，易于使用

使用限制

请注意，WebPortScanner 有一些使用限制，这是由于浏览器安全限制造成的。

  • 由于浏览器安全策略，无法进行某些高级扫描
  • 对于大型网络，扫描速度可能较慢
  • 无法检测某些防火墙后的端口

常见问题

以下是一些用户常见的问题和解答。

  • 可以扫描任何IP地址吗？是的，但请遵守相关法律法规
  • 扫描结果保存在哪里？保存在您的浏览器本地存储中
  • 可以导出扫描结果吗？是的，支持多种格式导出
